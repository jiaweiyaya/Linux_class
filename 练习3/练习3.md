# 第三题：圆形周长与面积计算 — 操作指南

> **注意：** 以下所有命令均在 `练习3/` 目录下执行。
> 如果当前不在该目录，请先执行：
```bash
`cd "练习3"`
```

---

## 第1部分：静态库方式（直接命令行编译）

### 步骤1：编译目标文件

```bash
gcc -Wall -g -c circlefuns.c -o circlefuns.o
```

### 步骤2：生成静态库 libmycircle.a

```bash
ar rcs libmycircle.a circlefuns.o
```

### 步骤3：编译测试程序并链接静态库

```bash
gcc -Wall -g test.c -L. -lmycircle -o test_static
```

### 步骤4：运行测试程序

```bash
./test_static
```

按提示输入半径（如 `5`），预期输出：

```
请输入圆的半径：5
圆的周长：31.42
圆的面积：78.54
```

---

## 第2部分：Makefile 方式

### 步骤1：清理之前生成的文件

```bash
make clean
```

### 步骤2：使用 Makefile 编译并链接（一步到位）

```bash
make test_static
```

> 该命令会自动编译 `circlefuns_static.o`、生成 `libmycircle.a`、然后链接生成 `test_static`。

### 步骤3：运行测试程序

```bash
./test_static
```

按提示输入半径（如 `10`），预期输出：

```
请输入圆的半径：10
圆的周长：62.83
圆的面积：314.16
```

---

## 第3部分：动态库方式

### 步骤1：清理之前生成的文件

```bash
make clean
```

### 步骤2：使用 Makefile 编译并链接（一步到位）

```bash
make test_dynamic
```

> 该命令会自动编译 `circlefuns_dynamic.o`（带 `-fPIC`）、生成 `libmycircle.so`、然后链接生成 `test_dynamic`。

### 步骤3：设置库搜索路径并运行

```bash
export LD_LIBRARY_PATH=.:$LD_LIBRARY_PATH
```
```bash
./test_dynamic
```

按提示输入半径（如 `7`），预期输出：

```
请输入圆的半径：7
圆的周长：43.98
圆的面积：153.94
```

---

## 附录：Makefile 常用命令

| 命令 | 说明 |
|------|------|
| `make static` | 仅编译静态库 `libmycircle.a` |
| `make test_static` | 编译静态库并生成测试程序（推荐） |
| `make dynamic` | 仅编译动态库 `libmycircle.so` |
| `make test_dynamic` | 编译动态库并生成测试程序（推荐） |
| `make all` | 编译静态库和动态库 |
| `make clean` | 清理所有编译生成的文件（`*.o`、`*.a`、`*.so`、`test_static`、`test_dynamic`） |
